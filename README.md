# JavaScript Recap – Kaikki osiot (1–5)

Tämä repositorio sisältää JavaScript-kurssin kaikki kertaustehtävät koottuna yhteen selkeään kokonaisuuteen.

## Tehdyt tehtävät ja pisteet

### Recap 1: Perusteet (11 p)
- **Assignment 1:** Lämpötilamuunnos (Celsius → Fahrenheit ja Kelvin)
- **Assignment 2:** Kahden pisteen välinen etäisyys 2D-tasolla
- **Assignment 3:** Kolmion tyypin määrittely sivujen pituuksien perusteella
- **Assignment 4:** Kurssiarvosanan laskeminen pistemäärän mukaan

### Recap 2: Taulukot ja funktiot (12 p)
- **Tehtävä 1:** Taulukoiden perusteet
- **Tehtävä 2:** Taulukko-operaatiot ja lajittelu
- **Tehtävä 3:** Parillisten lukujen poiminta

### Recap 3: DOM Assignments
- **Sijainti:** `/recap3/dom-assignments/`
- **Sisältö:** TODO-lista (insertAdjacentHTML & DOM Methods), BOM-rajapinnat, Geolocation-paikannus ja Leaflet.js-karttojen integrointi.

### Recap 4: Event Handling
- **Sijainti:** `/recap4/event-assignments/`
- **Sisältö:** Tapahtumankäsittelyn perusteet ja interaktiiviset kuuntelijat.

### Recap 5: AJAX & Restaurant App (34 p)
Tämä osio kattaa modernin tiedonhaun Reqres-rajapinnasta sekä Metropolian ravintola-API:n hyödyntämisen.
- **t1 & t2:** GET- ja POST-metodit API-avaimen kanssa.
- **t3:** Edistynyt virheidenhallinta (Try/Catch).
- **t4:** Yleiskäyttöisen `fetchData`-funktion kehittäminen.
- **t5:** Ravintolasovellus: hakee listan, järjestää aakkosittain ja näyttää päivän menun dynaamisessa modaalissa.

## Työkalut
- ESLint, Prettier, EditorConfig, Leaflet.js ja Fetch API.

## Käyttö
Avaa juurikansion **`index.html`** selaimessa siirtyäksesi tehtäviin.
